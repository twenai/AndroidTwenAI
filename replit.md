# Proyek WebView Android - Twen AI

## Informasi Aplikasi
- **Nama Aplikasi:** Twen AI
- **Package Name:** `com.meta.twenai`
- **SHA-1 Fingerprint:** `4C:D9:7C:EC:6B:74:28:01:EC:3E:24:11:79:03:CE:E5:64:04:3A:88`

## Fitur
- WebView ke https://kztutorial.com
- Support Google OAuth (Membuka browser eksternal untuk domain akun Google)
- GitHub Actions Workflow (`.github/workflows/android.yml`)

## Konfigurasi GitHub Actions
1. Push kode ini ke repository GitHub.
2. GitHub Actions akan otomatis menjalankan build.
3. Hasil APK (`app-debug.apk`) dapat diunduh dari tab **Actions**.

## Catatan Google OAuth
Gunakan SHA-1 di atas untuk mendaftarkan Client ID Android di Google Cloud Console agar fitur Login Google berjalan lancar.
